## 🚀 Le Projet Portfolio
Ce portfolio a été développé avec **Angular 16** et utilise un design épuré, dynamique et hautement personnalisé pour mettre en avant mon parcours professionnel et mes réalisations (notamment le projet majeur **ParabellumGroups ERP**).

### Structure du projet :

- `src/assets` : Contient l'ensemble des ressources statiques, des images optimisées, et les feuilles de style (CSS/SCSS).

## 🛠️ Comment exécuter le projet localement ?

Ce projet a été généré avec [Angular CLI](https://github.com/angular/angular-cli).

### Prérequis
- [Node.js](https://nodejs.org/) (version compatible avec Angular 16)
- npm (Node Package Manager)

### Installation
1. Clonez ce dépôt sur votre machine locale.
2. Naviguez dans le dossier du projet.
3. Installez les dépendances :
   ```bash
   npm install
   ```

### Serveur de développement
Exécutez la commande suivante pour lancer le serveur local :
```bash
npm start
```
Ou si vous avez Angular CLI installé globalement :
```bash
ng serve
```
Ouvrez ensuite votre navigateur et rendez-vous sur `http://localhost:4200/`. L'application se rechargera automatiquement si vous modifiez un des fichiers source.

## 📦 Compilation (Build)
Pour générer une version de production, exécutez :
```bash
ng build
```
Les fichiers compilés seront stockés dans le répertoire `dist/`.
